# Landing Solaire

Cette landing page est un simple fichier `index.html` (HTML + CSS). Aucun build n'est nécessaire :

## Visualiser la page
1. Clonez le dépôt puis ouvrez un terminal à la racine.
2. Ouvrez directement le fichier dans votre navigateur favori ou servez-le localement :
   ```bash
   # option A : ouvrir le fichier
   xdg-open index.html  # ou open index.html sur macOS

   # option B : servir via un mini serveur HTTP
   python3 -m http.server 8000
   # puis rendez-vous sur http://localhost:8000/index.html
   ```

La page est responsive et ne nécessite aucune dépendance supplémentaire.
